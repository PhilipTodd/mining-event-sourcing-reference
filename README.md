## Local Documentation

The project documentation is built using GitHub Pages and Jekyll.

### Prerequisites

- Ruby 3.2.x
- Bundler

### Run locally

```bash
cd docs
bundle install
bundle exec jekyll serve
```

Browse to:

```
http://localhost:4000
```
