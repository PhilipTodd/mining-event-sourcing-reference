# Mining Event Sourcing Reference

This repository contains a cloud-native event sourced
reference platform built using .NET 10 and Azure.

https://event-sourcing.ausdatatech.com.au

[Launch demo site](https://demo.event-sourcing.ausdatatech.com.au/){:target="_blank"}

## Local Documentation

The project documentation is built using GitHub Pages and Jekyll.

### Prerequisites

- Ruby 3.2.x
- Bundler

### Run locally

```bash
cd docs
bundle install
bundle exec jekyll serve
```

Browse to:

```
http://localhost:4000
```
